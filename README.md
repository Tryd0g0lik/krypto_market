## APP CryptoMarket
### Протокол
Приложение имеет временное ограничение в 1-ну минуту.\
Поэтому строится на HTTP протоколе. Изменилась или не изменилась цена, происходит запрос.

Если бы ориентация приложения была на изменение цены, то можно использовать WebSocket. 



Person чтоб платформа не:
     - зависила от от пользователей,
     - и не несла ответственности за действия пользователей


Данные об успешной авторизации кешируем на 27 часов. и сразу отправляю сообщение по SSE/ D cjj,otybb токены.
